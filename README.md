# Cocos2d-x로 만든 첫번째 게임

"내가 만든 이미지로 게임 만들기" 게임입니다.

사용자가 만든 이미지를 통해 카드 짝을 맞추는 게임입니다.

각 라운드마다 5초 안에 주어진 카드와 동일한 카드를 골라야 통과할 수 있으며,

총 3라운드로 진행됩니다.

이미지는 총 20개를 만들어야 실행이 가능합니다.
